Приложение поддерживает как xlsx, так и txt файлы на вход (данные по дням)
в формате где 1 колонка - дата, а 2 - температура

В моем готовом df данные по городу Москва с 2014 года\

Для запуска требуется python <=3.11

pip install -r requirements.txt
python console_app.py
